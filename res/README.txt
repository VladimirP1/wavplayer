Player 1.0 documentation
---------------------------

Проигрыватель поддерживает wav с целочисленным PCM содержимым.
Файл со звуком должен быть помёщён в корень флешки и назван x.wav
Его формат должен быть любым из целочисленных 8/16/24/32-бит PCM.
В файле screenshot.png выделены форматы, в которых можно сделать экспорт из Audacity.
Если создать файл inhibit.txt на флешке, то проигрыватель запускаться не будет.
Отладочная информация записывается в файл log.
Этот файл (README.txt) вместе с screenshot.png будет копироватся на USB-накопитель в момент запуска.
